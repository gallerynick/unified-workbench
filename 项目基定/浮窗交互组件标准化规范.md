# 浮窗交互组件标准化规范

> **常驻标准** — 所有浮窗（Modal/Drawer/Popover/Dropdown）内的交互组件必须遵守。
> 最后更新：2026-07-02

---

## 一、核心规则

**浮窗内部所有交互项禁止使用 `box-shadow`，仅允许通过以下两种反馈实现视觉高亮：**

1. **边框颜色变化**（`border-color`）— 参考 Input 组件的 focus 行为
2. **微放大效果**（`transform: scale`）— 固定倍率，不可自定义

---

## 二、实现标准（固定值）

### 交互卡片 / 选择卡片（如拓扑类型选择、模板选择等）

| 状态 | border-color | transform | background |
|------|-------------|-----------|------------|
| 默认 | `#d9d9d9` | `scale(1)` | `transparent` |
| 悬浮 (hover) | `#4096ff` | `scale(1.02)` | `transparent` |
| 选中 (selected) | `#1677ff` | `scale(1)` | `#e6f4ff` |

### 列表项（如拓扑列表、通知列表等）

| 状态 | border-color | transform | background |
|------|-------------|-----------|------------|
| 默认 | `#e8e8e8` | `scale(1)` | `transparent` |
| 悬浮 (hover) | `#4096ff` | `scale(1.01)` | `#f5f5f5` |
| 选中 (selected) | `#1677ff` | `scale(1)` | `#e6f7ff` |

### 深色模式映射

| 亮色值 | 深色值 |
|--------|--------|
| `#d9d9d9` | `#434343` |
| `#e8e8e8` | `#303030` |
| `#4096ff` | `#3c89e8` |
| `#1677ff` | `#1668dc` |
| `#e6f4ff` | `#111d2c` |
| `#f5f5f5` | `#2a2a2a` |
| `#e6f7ff` | `#111d2c` |

### 过渡动画（固定值）

```css
transition: border-color 0.2s, background-color 0.2s, transform 0.2s;
```

## 三、CSS 参考实现

### 选择卡片

```css
.selectCard {
  cursor: pointer;
  border: 1px solid #d9d9d9;
  border-radius: 8px;
  transition: border-color 0.2s, background-color 0.2s, transform 0.2s;
}

.selectCard:hover {
  border-color: #4096ff;
  transform: scale(1.02);
}

.selectCardSelected {
  border-color: #1677ff !important;
  background: #e6f4ff;
  transform: scale(1);
}

/* 深色模式 */
[data-theme="dark"] .selectCard {
  border-color: #434343;
}
[data-theme="dark"] .selectCard:hover {
  border-color: #3c89e8;
}
[data-theme="dark"] .selectCardSelected {
  border-color: #1668dc !important;
  background: #111d2c;
}
```

### 列表项

```css
.listItem {
  display: flex;
  align-items: center;
  justify-content: space-between;
  gap: 8px;
  padding: 12px;
  border: 1px solid #e8e8e8;
  border-radius: 6px;
  cursor: pointer;
  transition: border-color 0.2s, background-color 0.2s, transform 0.2s;
}

.listItem:hover {
  border-color: #4096ff;
  transform: scale(1.01);
  background: #f5f5f5;
}

.listItemActive {
  border-color: #1677ff;
  background: #e6f7ff;
  transform: scale(1);
}

/* 内容区列对齐 */
.listItemContent {
  flex: 1;
  min-width: 0;
  overflow: hidden;
}

/* 名称溢出省略 */
.listItemName {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}
```

## 四、TSX 使用规范

### 选择卡片

```tsx
<Card
  size="small"
  className={`${styles.selectCard} ${selected ? styles.selectCardSelected : ''}`}
  onClick={handleSelect}
>
  {/* 卡片内容 */}
</Card>
```

### 列表项

```tsx
<List.Item
  className={`${styles.listItem} ${active ? styles.listItemActive : ''}`}
  actions={[<Button danger icon={<DeleteOutlined />} />]}
  onClick={handleClick}
>
  <div className={styles.listItemContent}>
    <div className={styles.listItemName}>{name}</div>
    <Tag>{tag}</Tag>
  </div>
</List.Item>
```

## 五、禁止事项

| 禁止 | 原因 |
|------|------|
| ❌ `Card hoverable` 属性 | Ant Design 自动注入 `box-shadow` |
| ❌ 行内 `style={{ boxShadow }}` | 直接违反规则 |
| ❌ CSS `box-shadow`（装饰性） | 浮窗内禁止装饰性阴影 |
| ❌ 自定义 `transform` 倍率 | 必须使用固定值（卡片 1.02，列表 1.01） |
| ❌ 自定义 `transition` 时长 | 必须使用固定值 0.2s |

### 例外：功能性 Focus Ring

`box-shadow: 0 0 0 Npx rgba(...)` 模式（spread > 0, blur = 0, 无 X/Y 偏移）属于功能性 outline 模拟，不受禁止。

## 七、审查清单

所有浮窗内交互组件必须逐项通过以下检查：

- [ ] 无 `Card hoverable` 属性
- [ ] 无行内 `boxShadow`
- [ ] 无 CSS 装饰性 `box-shadow`
- [ ] 选中态使用 `border-color: #1677ff`（或深色 `#1668dc`）
- [ ] 悬浮态使用 `border-color: #4096ff`（或深色 `#3c89e8`）
- [ ] 卡片类组件悬浮态 `transform: scale(1.02)`
- [ ] 列表类组件悬浮态 `transform: scale(1.01)`
- [ ] 过渡动画 `0.2s`
- [ ] 深色模式已适配
