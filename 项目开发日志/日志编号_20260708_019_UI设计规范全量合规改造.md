#### P11 — 按钮圆角 CSS-in-JS 覆盖修复（global.css）

用户反馈工作台按钮（如"自定义布局"、"新建事件"）圆角仍为 pill 形状，非预期的 8px。

**问题1：CSS-in-JS 优先级覆盖**
- 根因：Ant Design 5 使用 CSS-in-JS（styled-components）生成 hash 类（如 `.css-12usjje`），其规则与 `.ant-btn` 同级（特异性均为 0,1,0），简单选择器无法覆盖
- 修复：使用 `#root` 前缀提高特异性 → `#root .ant-btn { border-radius: var(--rounded-sm) !important; }`（特异性 1,1,0）

**问题2：Docker 产物过期**
- 根因：Dockerfile 使用 `COPY dist /usr/share/nginx/html`，dist/ 构建于 14:22 但 global.css 修改于 15:38，容器内为旧 CSS
- 修复：重新 `npm run build` + `docker compose build --no-cache frontend` + `docker compose up -d frontend`

**问题3：Modal/Drawer Portal 渲染导致选择器不匹配**
- 根因：Ant Design Modal/Drawer 通过 React Portal 渲染到 `document.body` 下，不在 `#root` 内。`#root .ant-modal .ant-btn` 无法匹配
- 修复：去掉 `#root` 前缀 → `.ant-modal .ant-btn, .ant-drawer .ant-btn { border-radius: 9999px !important; }`
- 由于 Modal/Drawer 在 Portal 中，`.ant-modal .ant-btn` 与工作台 `#root .ant-btn` 作用域不重叠，无冲突

**最终效果：**

| 场景 | 圆角 | CSS 规则 | 特异性 |
|------|------|---------|:------:|
| 工作台普通按钮 | 8px | `#root .ant-btn` | 1,1,0 |
| 浮窗/模态框/抽屉按钮 | pill (9999px) | `.ant-modal .ant-btn, .ant-drawer .ant-btn` | 0,2,0 |
