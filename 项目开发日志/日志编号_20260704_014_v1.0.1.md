# v1.0.1 更新日志

## 模板系统修复（6 项）

1. **TemplateSelector 模板列表为空** — 修复 `listTemplates` 传 `location='project'` 导致全局模板被排除，改为前端过滤 `project|global`
2. **套用模板后编辑器内容不更新** — ContentEditor 改为 `forwardRef` + `useImperativeHandle` 暴露 `setContent()`，模板应用后通过 ref 直接注入内容
3. **buildTemplateContent 忽略 richtext 字段 content** — 模板的富文本内容（richtext 类型）存储在 `field.config` 中，原先只读取 `default_value` 导致内容丢失。修复后直接提取 `field.config.content` 嵌入文档
4. **模板应用后含多余标题** — 移除了模板名称 H1 标题和字段标签（如 H3「内容」），仅输出 richtext 正文
5. **选择模板按钮文案** —「套用模板」→「模板」
6. **模板列表去除位置标签** — 移除卡片上的「全局/项目/记录」标签

## 编辑器交互修复（1 项）

7. **空文档套用模板不弹确认** — 修复 `hasContent` 检查逻辑，复用列表视图已有的 `extractTextSummary` 函数判断文档是否为空，空文档直接应用模板

## 项目列表增强（1 项）

8. **项目列表新增「所有者」列** — Record 模型添加 `owner_name` 属性，schema 新增字段，前端表格显示创建者昵称

## 后端修复（1 项）

9. **记录所有者数据为空** — `list_records` 查询添加 `selectinload(Record.owner)` 显式加载 User 关联，确保 `owner_name` 不为空
