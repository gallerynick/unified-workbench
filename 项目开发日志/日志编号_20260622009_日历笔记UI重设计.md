### 开发内容详述

**1. 后端树形支持**
- Note 模型添加 `parent_id` 自引用外键（`ondelete="SET NULL"`），建立 `children`/`parent` relationship
- `create_note` 服务函数接收并传入 `parent_id`
- 新增 `list_all_notes`：不分页返回当前用户所有笔记（供前端构建完整树）
- 新增 `move_note`：支持移动笔记到新的父节点，含循环引用检查（不能移到自己或子孙节点下）
- API 层新增 `GET /notes/all` 和 `PUT /notes/{id}/move` 端点

**2. 日历 FullCalendar 重构**
- 使用 `@fullcalendar/react` + `daygrid` + `timegrid` + `interaction` 插件
- 支持月视图（dayGridMonth）、周视图（timeGridWeek）、日视图（timeGridDay）
- 使用 `events` 回调函数动态获取数据，配合 `startStr`/`endStr` 参数
- 新增表单字段：`end_time`（结束时间）、`all_day`（全天开关）、`location`（地点）、`color`（颜色选择器）、`repeat`（重复周期）
- 支持事件拖拽更新开始时间
- 中文本地化（`@fullcalendar/core/locales/zh-cn`）
- CSS 添加 FullCalendar 明/暗主题覆盖

**3. 笔记树形知识库重构**
- 使用 Ant Design `Tree` 组件替代 Table
- 从扁平列表构建层级树（`parent_id` 为 null 的是根节点）
- 支持拖拽排序和改变父节点（调用 `moveNote` API）
- 每个节点显示：图标（文件/文件夹）、置顶标记、分类标签、操作按钮（置顶/编辑/删除）
- 搜索过滤：按标题/内容/分类匹配，保留匹配的父路径
- 新建/编辑笔记时支持选择父笔记（TreeSelect 组件）
- 删除时检测子笔记并提示级联删除

### 遇到的问题与解决
- **问题 1**：TypeScript `exactOptionalPropertyTypes` 严格模式下，`undefined` 不能赋值给可选属性
  - 解决方案：修改类型定义，将 `?: string` 改为 `?: string | undefined`
- **问题 2**：Ant Design Tree `onDrop` 参数类型中 `dropNode` 应为 `node`
  - 解决方案：查阅 rc-tree 类型定义，使用 `info.node` 获取放置目标
- **问题 3**：后台任务（bg_78104787、bg_85e3ebc2）超时未完成
  - 解决方案：取消后台任务，直接接管并手动编写核心文件
- **问题 4**：FullCalendar CSS 未自动注入，需要手动导入
  - 解决方案：在组件中直接 import FullCalendar 相关包（CSS 通过 JS 模块自动加载）
