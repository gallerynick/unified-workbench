## 日志编号_20260708_018_按钮圆角规范修复

### 开发版本
版本号：v1.0.1

### 关联基定
《项目基定/UI设计规范.md》—— 按钮组件圆角规格（button-primary pill / button-dark-utility sm）+ 组件 borderRadius 全量核对

### 开发目标
1. 修复 Button borderRadius（default/small）从 8/4 → 9999（pill），使所有主按钮统一为胶囊形
2. 补充 Badge/Switch/Slider/Radio 缺失的 borderRadius token
3. Login 页面按钮 CSS 隔离为 button-dark-utility（8px）
4. 全量扫描所有组件 borderRadius vs 规范

### 涉及模块 / 文件
- 模块：主题与样式系统
- 修改文件：
  - `项目文件/frontend/src/contexts/ThemeContext.tsx` — Button borderRadius→9999, Button borderRadiusSM→9999, Badge borderRadius→4, Switch borderRadius→9999, Slider borderRadius→9999, Radio borderRadiusSM→9999
  - `项目文件/frontend/src/global.css` — 添加 login-page 按钮 8px 隔离
  - `项目文件/frontend/src/pages/Login.tsx` — 外层 div 添加 login-page 类名
  - `项目文件/frontend/src/pages/Login.module.css` — card borderRadius md→sm
- 新增文件：无
- 删除文件：无

### 开发内容详述

#### borderRadius 全量扫描结果（45 个组件 token 逐一对比）

| 组件 | 当前值 | 规范值 | 状态 |
|------|--------|--------|------|
| Button borderRadius | 8 | pill=9999 | ❌ → 已修复 |
| Button borderRadiusSM | 4 | pill=9999 | ❌ → 已修复 |
| Badge borderRadius | 未设置 | chip=4 | ❌ → 已添加 |
| Switch borderRadius | 未设置 | pill=9999 | ❌ → 已添加 |
| Slider borderRadius | 未设置 | full=9999 | ❌ → 已添加 |

#### 设计意图说明
- `button-primary` pill (9999) 是品牌标志性行动信号，所有主行动按钮必须统一
- `button-dark-utility` sm (8px) 仅用于登录/购物车/语言选择器
- `rounded.md` (11px) 珍珠按钮为稀有变体，不用于 Ant Design 默认按钮

- 解决方案：在 Login.tsx 外层 div 添加 `className="login-page"`
- 遗留问题：无
