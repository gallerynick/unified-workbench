# 间距标准化计划

## 背景

经全量扫描 35 个 CSS Module 和 16 个 TSX 文件中的间距使用，发现存在大量不一致：

- `.container` gap：大部分 16px，**2 个例外**（NoteManagement 20px、FormResponses 无 gap）
- `.header` gap：**仅 1 个文件定义**（FormResponses 12px），其余 34 个无 header gap
- 内联样式 gap：4/6/8/12/16/24/40px 混用，无规律
- 内联 marginBottom：4/8/12/16/24px 混用
- Header 无统一 min-height 标准

## 一、标准间距定义

| 级别 | 值 | 用途 |
|------|-----|------|
| xs | 4px | 图标与文字间距、Tag 间距、紧密排列 |
| sm | 8px | 移动端 padding、行内相关元素、Space 默认 gap |
| md | 12px | 筛选器间距、Form.Item 间距、卡片内 padding |
| lg | 16px | 容器 gap、header gap、区块间距、桌面默认间距 |
| xl | 24px | 页面级 padding、卡片间、Alert 间距 |
| xxl | 48px | Header 最小高度 |

---

## 二、CSS Module 规范

### 2.1 标准页面容器

```css
.container {
  display: flex;
  flex-direction: column;
  gap: 16px;                           /* lg */
}

@media (max-width: 767px) {
  .container { padding: 8px; }         /* sm */
}
```

### 2.2 标准页面 Header

```css
.header {
  display: flex;
  align-items: center;
  justify-content: space-between;
  gap: 16px;                           /* lg */
  min-height: 48px;                    /* xxl */
}

.title {
  margin: 0 !important;
  font-weight: 600;
  white-space: nowrap;
}
```
