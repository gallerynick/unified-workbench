# 导入组织标准化计划

## 背景

扫描 68 个 TSX + 2 个 TS 文件，514 条 import。发现 5 类问题，涉及 10 个文件。

---

## 一、标准导入顺序

```
1. React / React Router（框架）
2. 第三方 UI 库（antd / antd-icons）
3. 其他第三方（@dnd-kit / dayjs / FullCalendar）
4. 本地 API / hooks / contexts / utils
5. 类型导入（import type）
6. 本地组件
7. 样式（CSS Module / 全局 CSS）
8. 资源（SVG / 图片）
```

---

## 二、问题清单

| 优先级 | 问题 | 文件 | 修复 |
|--------|------|------|------|
| P0 | 重复 antd Result 导入 | SiteSettings, SidebarManagement | 合并到主 import |
| P1 | 导入顺序错乱 | SystemSettings, FormManagement, QuickLinksWidget, TopologyManagement, CalendarPage | 调整顺序 |
| P2 | 同模块类型导入分多行 | BackupManagement, ReminderFormModal | 合并为一行 |
| P3 | 内联 type 风格不统一 | SystemSettings, GraphView, Home | 统一用 import type |
| P4 | 缺失 barrel 导出 | Home(widgets), ProjectDetailPage(tabs) | 可选优化 |

---
