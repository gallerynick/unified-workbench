### 2. 创建推流拉流页面

**StreamStudio.tsx 核心功能：**

- **场景系统**：多个场景，每个场景包含多个源
- **源类型**：
  - 本地摄像头（getUserMedia）
  - 共享屏幕（getDisplayMedia）
  - 网络流（HLS/RTSP via video element）
  - 图片/文字叠加
- **预览区**：实时预览当前场景
- **推流**：通过 MediaRecorder + WebSocket 推流到服务器
- **拉流**：播放网络流地址

### 3. 创建后端流媒体 API

- `GET /api/v1/stream/config` — 获取推流配置
- `PUT /api/v1/stream/config` — 设置推流地址
- `POST /api/v1/stream/start` — 开始推流
- `POST /api/v1/stream/stop` — 停止推流
